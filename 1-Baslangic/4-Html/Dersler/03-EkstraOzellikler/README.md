# HTML Etiketleri

## Yaygın Kullanılan Semantik Olmayan Etiketler

- h1 - Başlık

- p - Paragraf

- i / em - Eğik / Vurgulu Yazı

- b / strong - Kalın / Koyu Yazı

- a - Link

- ul & li  - Sırasız liste ve liste elemanı

- blockquote - Alıntı

- hr - Yatay çizgi ekleme

- img - Görsel

- div - Bölme
